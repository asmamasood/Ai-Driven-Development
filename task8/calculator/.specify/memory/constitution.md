The purpose of this calculator is to allow users to input two numbers and choose an operation, then receive an accurate and clear output.

To create a reliable, user-friendly, and error-free calculator using the 5 AI-driven development phases of SPECKitPlus.

This project will include:

* Input handling
* Arithmetic operations
* Clean console output
* Error handling

* The calculator must accept **two numeric inputs**.

* Division by zero must be safely handled.
* Code must be simple, readable, and beginner-friendly.

* The calculator runs without errors.
